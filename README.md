# Bank-Account-System-IEEE
Bank Account System mini-project developed for MTI IEEE.

## Introduction
A Python OOP exercise written for an IEEE assignment to demonstrate a fundamental understanding of Object-Oriented Programming principles.

## Overview
The program begins by verifying user credentials against a basic database structured as a dictionary of dictionaries. There are two primary classes representing user types: **User** and **Admin**. 

The **Admin** class inherits all methods from the **User** class, allowing for a hierarchical structure. 
* **Users** can check their balance, deposit funds, and withdraw money. 
* **Admins** possess additional privileges, including the ability to ban existing users and create new accounts.
